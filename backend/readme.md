Il faudra avoir installé:
    * Node.js (et donc npm),
    * MySQL,

il faut faire un "npm install" puis un "nodemon server" dans "/backend" pour lancé le serveur

et un "npm install" puis "npm start" dans "/backend/client" pour lancer le front
